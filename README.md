A full-stack social network simulation built as a team project.
Includes authentication, feed posting, social interactions, and user discovery.

## Repositories
- **Client (React):** https://github.com/AsafRez/SocialNetworkApp
- **Server (Java):** https://github.com/AsafRez/Social_Server

## Features
- Register / Login
- Feed: create posts
- Likes / Dislike
- Comments
- Search users
- Follow / Friends
- Change profile picture
- Upload image

## Tech Stack
- **Client:** React, JavaScript
- **Server:** Java
- **Data:** local Java-based storage (initialized per environment)
- **Tooling:** Git

## How to Run (Local)
### Server (Java)
1. Open the server project in IntelliJ
2. Run the server (local storage is initialized on first run)

### Client (React)
1. `npm install`
2. `npm start`
3. Open `http://localhost:3000`
